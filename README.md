# teach_vue

> [!IMPORTANT]
>
> 项目名称 teach_vue

#### 安装环境

1. 下载nodejs1.1.12并安装

   setup.exe：https://github.com/coreybutler/nvm-windows/releases/download/1.1.12/nvm-setup.exe
2. 在命令提示符中端中输入命令

   ```shell
   nvm install 20
   ```

   ```shell
   nvm use 20.19.1
   ```

   检查是否安装成功

   ```shell
   node -v
   npm -v
   ```


## 克隆 & 运行项目

> [!TIP]
>
> 也可以参考以下步骤在本地运行项目

#### 1. 克隆项目到本地

```
git clone https://github.com/Ethan-Zhengsy/teach_vue.git
cd teach_vue
```

#### 2. 安装依赖

```
npm install
```


#### 3. 启动项目

```
npm run dev
```


#### 4. 访问网页

 一般情况下，直接 `Ctrl + 左键单击`即可打开页面。如果打不开，可以尝试把网址复制到浏览器地址栏中。
